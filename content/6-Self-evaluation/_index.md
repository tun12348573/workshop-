During my internship at **AWS-FCAJ** from **April 17, 2026** to **July 10, 2026**, I had the opportunity to learn, practice, and apply the knowledge I gained at university in a real working environment.

I participated in the **Mimi Jewelry E-commerce Project**, through which I improved many important skills such as frontend development, backend development, AWS cloud deployment, requirement analysis, debugging, technical documentation, teamwork, and progress reporting.

Regarding my working attitude, I always tried to complete assigned tasks properly, follow company rules, and actively communicate with colleagues to improve work efficiency.

#### Skills gained during the project

Through the **Mimi Jewelry E-commerce Platform** project, I learned and practiced the following skills:

- **Frontend development skills**: building an e-commerce website interface using Next.js, including product pages, cart, checkout, customer login, and admin dashboard.
- **Backend development skills**: building APIs for products, orders, payments, customer accounts, and admin functions using Node.js/TypeScript on AWS Lambda.
- **AWS Serverless skills**: using services such as AWS Lambda, API Gateway, DynamoDB, S3, CloudFront, Cognito, EventBridge, CloudWatch, SNS, and SES.
- **Infrastructure as Code skills**: using AWS SAM and CloudFormation to define, build, and deploy backend resources to AWS.
- **Database management skills**: designing and using DynamoDB to store products, orders, payment status, and refund information.
- **Authentication and authorization skills**: using Amazon Cognito to separate customer and admin login, and protecting APIs with JWT Authorizer.
- **File upload handling skills**: creating presigned URLs to upload product images to Amazon S3 and distributing images through CloudFront.
- **Payment integration skills**: integrating ZaloPay Sandbox for checkout, creating payment URLs, handling payment callbacks, and updating order status.
- **Monitoring and debugging skills**: using CloudWatch Logs to debug Lambda functions, API Gateway requests, payment callbacks, and scheduled jobs.
- **System alerting skills**: configuring CloudWatch Alarms and SNS to receive email alerts when errors or abnormal metrics occur.
- **Basic CI/CD skills**: understanding the deployment process for frontend and backend, and using GitHub/GitHub Actions during development.
- **Basic AWS security skills**: understanding IAM Roles, Policies, CORS, JWT tokens, and access control for cloud resources.
- **Functional testing skills**: testing main flows such as registration, login, product management, ordering, payment, and order management.
- **Technical documentation skills**: creating workshop content, describing architecture, deployment steps, screenshots, and cleanup instructions.
- **Teamwork and communication skills**: communicating with mentors, receiving feedback, reporting progress, and collaborating to complete project tasks.

### Areas for improvement

Besides the skills I gained, I also recognize that I still have several areas that need improvement in the future:

- **Improve discipline**: I need to follow working hours, company rules, and working procedures more strictly in any organization.
- **Improve problem-solving thinking**: I need to practice analyzing root causes, breaking down problems, and finding solutions in a more systematic way.
- **Improve communication skills**: I need to present ideas more clearly, communicate actively when facing difficulties, and respond better during teamwork.
- **Improve time management**: I need to organize tasks by priority and avoid leaving important work until close to the deadline.
- **Improve English technical reading skills**: Since many AWS and framework documents are written in English, I need to read and understand technical documents faster and more accurately.
- **Improve debugging skills**: I need to improve my ability to read logs, identify errors, and check systems in a more structured way.
- **Improve knowledge of system security**: I need to learn more about IAM, API authorization, user data protection, and best practices for deploying applications on the cloud.
- **Improve report writing skills**: I need to present content more clearly, use suitable screenshots, and describe the project implementation process more effectively.
- **Be more proactive in asking questions and receiving feedback**: I need to communicate more confidently with mentors or colleagues when facing issues to save time and learn faster.
- **Continue improving technical knowledge**: I need to study backend, frontend, cloud deployment, CI/CD, and system design more deeply so that I can contribute better to real projects.

Overall, the internship helped me better understand the real working environment, the process of deploying a cloud-based e-commerce project, and the skills required of a software developer. This experience is an important foundation for me to continue improving myself and developing my career path in software engineering and cloud computing.
