### Week 3 Objectives:

- Learn about serverless backend, monitoring, and CI/CD on AWS.

### Tasks to be completed this week:

| Day | Task                                                                                                                                        | Start date | Completion date | References                                                                     |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------- | ---------- | --------------- | ------------------------------------------------------------------------------ |
| 2   | - Learn AWS Lambda, understand function, handler, runtime, environment variables, and how Lambda handles requests without managing servers. | 04/05/2026 | 04/05/2026      | https://aws.amazon.com/vi/lambda/                                              |
| 3   | - Learn Amazon API Gateway, understand HTTP API, route, method, endpoint, and how API Gateway invokes Lambda.                               | 05/05/2026 | 05/05/2026      | https://aws.amazon.com/vi/api-gateway/                                         |
| 4   | - Learn Amazon EventBridge, understand rule/schedule and how to trigger Lambda on a schedule for background tasks.                          | 06/05/2026 | 06/05/2026      | https://docs.aws.amazon.com/eventbridge/latest/userguide/eb-what-is.html       |
| 5   | - Learn about Amazon CloudWatch and Amazon SNS. Learn how to write logs, monitor metrics, create alarms, and send alerts via email.         | 07/05/2026 | 07/05/2026      | https://docs.aws.amazon.com/sns/latest/dg/sns-monitoring-using-cloudwatch.html |
| 6   | Learn about AWS SAM, CloudFormation, and GitHub Actions. Learn how to manage infrastructure using templates and prepare for CI/CD.          | 08/05/2026 | 08/05/2026      | https://aws.amazon.com/vi/serverless/sam/                                      |

### Results achieved in Week 3:

- Gained an understanding of the main services used to build a serverless backend, monitoring, alerting, and automated deployment process, such as:

* AWS SAM
* Amazon API Gateway
* Amazon EventBridge
* AWS Lambda
* Amazon CloudWatch and Amazon SNS
